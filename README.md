## Explication du code
Ce projet implémente une fonction de validation d'Email en TypeScript (TS) sans utiliser de fonctions préconçues. Le développement a été réalisé en suivant la méthodologie TDD (Test Driven Development). La fonction vérifie que la chaîne contient au moins un '@', un point dans le nom de domaine (qui n'est pas le dernier caractère), aucun espace, et qu'il y a du texte avant et après le '@'.
